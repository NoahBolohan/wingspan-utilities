2. Deploying a Google Web App (b)

  • In the top right, click on Deploy > New deployment
  • Under the Select type > cogwheel, choose Web app
  • Customize the fields as you wish and click Deploy (you may have to authorize permissions again)
  • If succesful, you should be presented with a pop-up that contains a Deployment ID and a URL. This URL is your personal Web App URL to be used in the Wingspan Utilities application
